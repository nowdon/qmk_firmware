# The MS Excel keymap for quick7

### Base Layer
```
Left Encoder(CW: DOWN, CCW: UP)
Right Encoder(CW: RIGHT, CCW: LEFT)
,----------- ---------- ------------,
|   Left    |  Func1   |   Right    |
|  Encoder  |          |  Encoder   |
 ----------- ---------- ------------
|  PASTE    |  PASTE   |   PASTE    |
|  VALUE    | FORMULA  |   FORMAT   |
 ----------- ---------- ------------
|  SELECT   |    E     |   ENTER    |
|    ALL    |          |            |
`----------- ---------- ------------'
```

### Func1 Layer
```
,----------- ---------- ------------,
|   Left    |  TRNS    |   Right    |
|  Encoder  |          |  Encoder   |
 ----------- ---------- ------------
|  INSERT   |  DELETE  |     F4     |
|   LINE    |   LINE   |            |
 ----------- ---------- ------------
|   MERGE   |  SPLIT   |   ENTER    |
|   CELLS   |  CELLS   |            |
`----------- ---------- ------------'
