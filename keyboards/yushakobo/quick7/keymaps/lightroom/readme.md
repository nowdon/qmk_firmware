# The lightroom keymap for quick7

### Base Layer
```
Left Encoder(CW: UP, CCW: DOWN)
Right Encoder(CW: RGB HUE-, CCW: RGB HUE+)
,----------- --------- ------------,
|   MUTE    |  Func1  |  RGB MODE  |
 ----------- --------- ------------
| Shift+Tab |   UP    |    TAB     |
 ----------- --------- ------------
|   LEFT    |  DOWN   |   RIGHT    |
`----------- --------- ------------'
```

### Func1 Layer
```
,----------- --------- ------------,
|   RESET   |  Func1  | RGB TOGGLE |
 ----------- --------- ------------
|   HOME    |  VOL+   |    END     |
 ----------- --------- ------------
| MEDIA <<  |  VOL-   |  MEDIA >>  |
`----------- --------- ------------'
```
